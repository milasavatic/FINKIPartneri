# FINKIPartneri

**FINKIPartneri** is a simple frontend-only project built using HTML, CSS, and Bootstrap.  
The goal of the project is to demonstrate responsive design, layout structure, and UI components for a FINKI's partner-listing website.

## Features

- Fully responsive layout using **Bootstrap**
- Custom styles written in CSS  
- Example UI components:
  - Navigation bar
  - Cards / Partner previews
  - Footer
- Clean and modern frontend design
